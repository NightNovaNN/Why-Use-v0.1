# Why-Use-v0.1
Part of the Why Framework, tells you why to use after index

## Why?
It is useful when we need to find a particular word very fast. It indexes a given directory and when searched, it can show which files and lines.

---

## Usage

### Download
- Using Git:
  ```sh
  git clone https://github.com/NightNovaNN/Why-Use-v0.1
  cd Why-Use-v0.1
  ```
- From Github Fork:
  Click the `fork` button to fork the repo

### Use
- Index:
  ```sh
  python wu.py index dir/
  ```
- Search:
  ```sh
  python wu.py search term
  ```
- Clear:
  ```sh
  python wu.py clear
  ```
